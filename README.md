# 🎮 MonCon

**MonCon** is a lightweight Windows utility that monitors Xbox controller connection status and battery level in real time.

Designed for simplicity, speed, and clarity — plug in your controller and instantly see its status.

---

## 🌙 Preview

> Clean, minimal controller monitoring dashboard with live updates.

---

## ✨ Features

### 🎮 Controller Detection
- Automatically detects Xbox controllers
- Supports **USB and Bluetooth**
- Real-time connection updates

### 🔋 Battery Monitoring
- Live battery percentage display
- Updates automatically while connected
- Simple visual battery indicator

### ⚡ Real-Time Status
- Instant connect/disconnect detection
- No manual refresh required
- Lightweight background update system

### 🧠 Simple UI
- Minimal, distraction-free design
- Fast performance
- Plug-and-play usage

---

## 🖥️ Requirements

- Windows 10 / 11
- Xbox controller (Xbox One / Series S|X)
- USB or Bluetooth connection
